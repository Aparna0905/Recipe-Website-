Recipe Website 
Overview:
This is a responsive Recipe Website created using HTML and CSS.  
The project focuses on clean UI design, structured layouts, and visually appealing food sections to enhance user experience.
 Features
- Navigation bar with multiple sections
- Featured recipe section (Gulab Jamun)
- Recipe cards with images and titles
- Popular categories section
- Newsletter subscription form
- Footer with quick links and social media
- Clean and user-friendly UI layout
Technologies Used
- HTML5
- CSS3
